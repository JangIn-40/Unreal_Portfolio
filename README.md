## 1. 프로젝트 소개
- GAS를 활용한 언리얼 엔진 포트폴리오
- 제작 기간 : 2025.10.23 ~ 2025.12.01 (약 한달)
- 개발 인원 : 1인
- Paragon 에셋을 활용해서 Wukong 영웅으로 몰려오는 적들을 처치하는 액션게임을 만들었습니다.
